ISMR

In-School Mobile Robot

ISMR developed as a term project in Gebze Institute of Technology.

What Is ISMR?

ISMR is a mobile robot which can go predefined destinations in a well-known building. The route is calculated by the client computer and sent to robot over wifi. Robot drives itself to destination.

ISMR sends a live video stream using RaspiCam. Stream can be played by any stream supported video player (like VLC). Client computer can see the stream with augmented text which indicates the robot's current location.

ISMR uses Raspberry PI for both driving robot, communication and video streaming. ISMR can automatically connect preconfigured wifi networks and support roaming inside a building.

ISMR project is completely open source.
Both Client and Rover's codes are available at our GitHub Repo!
